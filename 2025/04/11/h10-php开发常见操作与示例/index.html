H10 PHP 开发常见操作与示例

本文档整理了详细说明了 PHP 开发中的基础语法、数据库查询、迁移管理、消息队列管理、模型查询、文件处理以及代码规范检查等内容，包含具体代码示例和中文说明，适合快速参考和学习。

基础语法

字符串操作

explode 和 implode

用于字符串分割与合并，类似于 Python 的 split 和 join。

说明：

  • explode 将字符串按指定分隔符分割成数组，适用于解析固定格式的字符串。
  • implode 将数组元素拼接成字符串，常用于生成特定格式的输出。

str_replace 和 trim

字符串替换与去除首尾空白。

说明：

  • str_replace 支持批量替换，常用于文本清理或格式化。
  • trim 可指定去除的字符集，如 trim($text, " \t\n\r")。

输出与换行

说明：
PHP_EOL 是 PHP 内置的换行符常量，跨平台兼容（Windows 为 \r\n，Unix 为 \n）。

数组定义

说明：
PHP 数组支持键值对，类似 Python 的字典，常用于配置或数据传递。

三元表达式

简洁的条件判断语法。

说明：
三元表达式适合简单的条件赋值或输出，避免冗长的 if-else。

JSON 编码

说明：
json_encode 将 PHP 数组或对象转为 JSON 字符串，常用于 API 数据返回。

文件读写

说明：

  • file_get_contents 读取整个文件到字符串，适合小型文件。
  • file_put_contents 写入内容到文件，覆盖原有内容，支持追加模式（FILE_APPEND）。

HTML 解码

说明：
html_entity_decode 将 HTML 实体（如 &lt;）转换为对应字符，适用于显示用户输入或外部数据。

正则表达式

preg_match 和 preg_match_all

说明：

  • preg_match 匹配一次，返回第一个结果。
  • preg_match_all 匹配所有结果，返回数组，常用于批量提取。

数组操作

array_map

对数组每个元素应用回调函数。

说明：
array_map 适合批量转换数据，回调函数支持匿名函数（PHP 7.4+）。

array_filter

过滤数组元素。

说明：
array_filter 根据回调函数的返回值保留元素，适用于数据筛选。

in_array

检查元素是否存在于数组。

说明：
in_array 支持严格模式（strict => true），可检查类型和值。

Migrate 管理和使用

数据库迁移用于管理表结构变更，确保开发与生产环境一致。

1. 创建迁移文件

在 Docker 容器中运行命令生成迁移文件。

说明：

  • 命令格式：migrate/create <迁移名称>。
  • 迁移名称应清晰描述变更内容，如添加字段、修改表结构等。

2. 编辑迁移文件

生成的文件位于项目迁移目录，需实现 up 和 down 方法。
