Claude Code 逆向MCP 工具配置指南

前言

MCP（Model Context Protocol）允许 Claude Code 调用本地工具和服务。本文记录两个常用的 MCP 工具配置：JSReverser-MCP（JavaScript 逆向分析）和 Chrome DevTools MCP（浏览器控制）。

1. JSReverser-MCP

JavaScript 代码逆向分析工具，适合反混淆和代码分析场景。

2. Chrome DevTools MCP

通过 Chrome DevTools Protocol 直接控制浏览器，可替代部分 Playwright/Puppeteer 场景。

添加到 Claude Code

适用范围：npx 启动的包，最新版直接 claude mcp add 即可，无需手动编辑 JSON。

常用参数

参数 说明
--autoConnect 连接到已运行的 Chrome 实例
--headless 无 UI 模式运行
--slim 基础 3 工具模式
--isolated 使用临时配置文件，自动清理
--browser-url 连接指定 URL 的 Chrome
--channel 使用 canary/beta/dev 频道

启用 Autoconnect

  1. Chrome 中打开 chrome://inspect/#remote-debugging
  2. 勾选 Enable remote debugging
  3. 确保 Chrome 正在运行

配置文件位置（了解即可）

手动编辑时配置文件位置为 ~/.claude/mcp.json，但使用 claude mcp add 命令会自动写入，无需手动干预。
